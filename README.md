# IT Asset Management System

## Overview
Full-stack enterprise-style asset registry system for tracking IT equipment.

## Features
- Authentication (JWT)
- Asset CRUD
- Activity logs
- Responsive UI (mobile + desktop)
- Dark/Light mode
- Dashboard analytics

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Deployment: Vercel + Render
